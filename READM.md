## Overview

This project performs **molecular docking** of two natural flavonoid ligands — **Kaempferol** and **Myricetin** — against a target protein receptor using **AutoDock Vina**. The goal is to evaluate the binding affinity of these compounds and identify potential drug candidates.

---

## Ligands

| Ligand | Type | Description |
|---|---|---|
| **Kaempferol** | Flavonol | Natural polyphenol found in many plants; known anti-inflammatory and antioxidant properties |
| **Myricetin** | Flavonol | Polyphenolic compound with reported anticancer and antioxidant activities |

---

## Receptor

| File | Description |
|---|---|
| `clean_2063.pdbqt` | Prepared receptor structure (cleaned PDB → PDBQT format) |

The receptor was cleaned by removing water molecules (H₂O), hydrogen atoms, and heteroatoms, and fixing atomic connectivity using PyMOL and a text editor.

## Setup & Usage

### Prerequisites

Install [AutoDock Vina](https://vina.scripps.edu/downloads/):

```bash
# Ubuntu/Debian
sudo apt-get install autodock-vina

# Or download binary from https://vina.scripps.edu/downloads/
```

### Run docking

```bash
git clone https://github.com/your-username/molecular-docking-vina.git
cd molecular-docking-vina
bash script.sh
```

Results will be saved automatically in `results/Kaempferol/` and `results/Myricetin/`:
- `*_output.pdbqt` — docked poses
- `*_log.txt` — binding affinity scores for each mode

---

## Output Interpretation

AutoDock Vina ranks docked poses by **binding affinity (kcal/mol)**. Lower (more negative) values indicate stronger predicted binding:

```
mode |   affinity | dist from best mode
     | (kcal/mol) | rmsd l.b.| rmsd u.b.
-----+------------+----------+----------
   1 |       -7.5 |    0.000 |    0.000
   2 |       -7.2 |    1.823 |    2.541
```
